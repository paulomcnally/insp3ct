    • Insp3ct

    Identify intruders change files in git projects

    Install 

            
$ wget https://github.com/paulomcnally/insp3ct/raw/master/insp3ct_1.0_all.deb
$ sudo dpkg -i insp3ct_1.0_all.deb

    Configuration 

            
$ sudo vi /etc/default/insp3ct.cfg
